### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户（员工）、维修员、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 材料信息管理
6. 出入库信息、出入库列表信息
7. 产品信息管理
8. 顾客订单管理
9. 生产计划管理
10. 设备信息管理
11. 设备检修信息
12. 质量监控管理


**维修员功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 设备信息
6. 设备检修管理



**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 管理员管理
4. 维修员管理
5. 员工管理
6. 材料信息管理
7. 出入库管理、出入库列表管理
8. 产品信息管理
9. 顾客订单管理
10. 生产计划管理
11. 设备信息管理
12. 设备检修管理
13. 质量监控管理
14. 基础数据管理（部门管理、材料类型管理、产品类型管理、设备类型管理）
